# projet-tracabilite-karite
systeme de suivi numérique des lots de beurre de karite-IST Ouaga 2000-2025-2026
membres: tonde hortence ,zoungrana habiba, yougbare bernard
Ce projet porte sur la conception d'un système numérique dédié à la traçabilité des lots de beurre de karité, un produit stratégique pour l'économie locale. L'objectif est de documenter chaque étape du processus, depuis la collecte des noix jusqu'à la transformation finale, afin de garantir le respect des normes de qualité internationales.
